Chemy Board Game

Chemy Board Game is a trading card game inspired by Magic the Gathering mechanics. In game all of the cards are related to chemistry. Chemical elements retain their properties - for example, gases have a "gas" effect that acts as "flying" effect in MTG. There are over 120 playable cards.

Game purpose

  • Teaching Chemistry
  • Entertainment
