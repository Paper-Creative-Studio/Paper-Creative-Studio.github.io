Vabanque Hoster

Vabanque Hoster is a device that help you host an event like "Vabanque" TV show. Vabanque Hoster is made of 2 parts - questions displayer and hoster device. Displayer is a php application that reads CSV file and based on questions and premium points in that file displays them and count teams points. Hoster Device is based on ESP8266 microcontroller that hosts Wi-Fi network and monitors when connected to it players are volunteering to answer. This kind of TV show is used in history class insted of regular exams.

Device purpose

  • Hosting events
  • Determines which team has volunteered first
