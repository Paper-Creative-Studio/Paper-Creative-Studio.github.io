Python Defence

Python Defence is a 2D defence game. Your task is to keep your village safe. At start you have to help local villagers rebuild their workplaces after monster attack. Then you can upgrade you stats and village. To complete tasks you have to use a Python programming language and code small program. Dificulty of those grow in time.
